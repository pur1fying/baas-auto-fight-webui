# BlueArchive AutoFight WorkFlow Hub

## 概述
BAAS自动战斗相关前端界面, 旨在提供一个用户友好的蔚蓝档案战斗轴描述界面, 简化自动战斗轴文件的生成流程, 并为轴文件的分发, 查询, 储存提供一个平台, 由于业务流程与github相似 (轴文件仓库)

相关链接:

1. [BAAS自动战斗](https://baas.wiki/develop_doc/script/auto_fight.html)
2. [BAAS_Cpp](https://github.com/pur1fying/BAAS_Cpp) : 解读轴文件并执行的C++程序
3. [blue_archive_auto_script](https://github.com/pur1fying/blue_archive_auto_script)
